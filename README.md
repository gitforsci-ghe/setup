# README

This repository guides you through the initial setup and configuration of Git
for R projects, using the `usethis` package. Work through the checklist:

- [ ] Open the project in RStudio by opening `01-setup.Rproj`
- [ ] Install the `usethis` package: `install.packages("usethis")`
- [ ] Edit `git-configuration.qmd` and replace the two placeholders with your
      full name and the email address you use on GitHub
- [ ] Render the document and check the `usethis::git_sitrep()` output for
      your name, email, and personal access token
- [ ] Commit `git-configuration.qmd` and `git-configuration.html`, then push
      your changes to GitHub
- [ ] Open an issue in this repository titled "Completed test setup tasks"
      and tag @larnsce so the instructor can verify your setup
